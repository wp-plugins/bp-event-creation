== Description ==

This BuddyPress plugin allows members to create, edit and delete Events from their profile. Event creator can also check who will attend his event.

* creates a custom post type called 'event'.
* provides a tab on each members' profile for front-end creation, editing and deletion.
* event creator can check who will attend his event.
* any user can attend event.
* user can also upload image with his event. 

== Installation ==

1. Unzip and then upload the 'bp-event-creation' folder to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to Settings -> Viva Bp Events and select which user roles are allowed to create Events. 
Admins are automatically given permission.   Other settings are also available.
